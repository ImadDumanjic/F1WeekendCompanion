## How to take screenshots

Run the app locally with `npm run dev`, then visit each URL below and take a screenshot:

| Screen | URL |
|---|---|
| Login | http://localhost:5173/login |
| Register | http://localhost:5173/register |
| Dashboard | http://localhost:5173/ |
| Schedule | http://localhost:5173/schedule |
| Schedule detail | http://localhost:5173/schedule (click any race card) |
| Predictions | http://localhost:5173/predictions |
| Leaderboard | http://localhost:5173/leaderboard |
| AI Summary | http://localhost:5173/ai-summary |
| Profile | http://localhost:5173/profile |
| Select Favorite Driver | http://localhost:5173/select-driver |
